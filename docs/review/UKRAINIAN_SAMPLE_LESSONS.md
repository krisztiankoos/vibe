# Ukrainian Sample Lessons for v1.1.0

Complete TypeScript templates for 7 Ukrainian sample lessons: 3 GPPC (A1-B1 grammar-focused) + 4 CEFR (B1+ task-focused).

## Overview

These lessons demonstrate:
- GPPC structure for grammar instruction (deductive approach)
- CEFR structure for communicative tasks
- Bilingual explanations (Ukrainian + English)
- All 11 exercise types
- Pedagogically sound Ukrainian as Foreign Language (UFL) methodology

**Target users**: Ukrainian teachers teaching Ukrainian to foreigners
**Content**: Completely different from English lessons (not translations)
